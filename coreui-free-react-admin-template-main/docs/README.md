# Project Documentation

This directory contains all documentation for the CoreUI React Admin Template customization.

## Directory Structure

- `pages/` - Detailed documentation for each page/feature
- `decisions/` - Important architectural and technical decisions
- `meeting-notes/` - Notes from discussions and planning sessions
- `CHANGELOG.md` - Record of all notable changes to the project

## Documentation Guidelines

1. Each major feature/page should have its own markdown file
2. Keep documentation up-to-date with code changes
3. Use clear headings and consistent formatting
4. Include code examples where helpful
5. Document both what was done and why it was done that way
